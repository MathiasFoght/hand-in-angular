## Angular - Credit Card Management System
### How to run
1. Navigate to the project directory:
   ```bash
   cd credit-card-frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
    ```bash
    ng serve
    ```
4. You are ready!
